# langtune

langtune is a Python package for fine-tuning large language models (LLMs) on text tasks. It provides modular components, LoRA-based parameter-efficient adaptation, and a simple interface for research and production use.

## Features
- LoRA adapters for efficient fine-tuning
- Modular transformer backbone
- Model zoo for open-source language models
- Configuration system and CLI
- Checkpointing and resume support
- Distributed and mixed-precision training

## Installation

```bash
pip install langtune
```

## Basic Usage

```python
import torch
from langtune.models.llm import LanguageModel

input_ids = torch.randint(0, 1000, (2, 128))
model = LanguageModel.from_pretrained("transformer_base")
output = model(input_ids)
```
